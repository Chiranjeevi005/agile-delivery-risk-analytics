# Data Dictionary

## Jira Export Data
| Field Name | Data Type | Description |
| :--- | :--- | :--- |
| `Issue Key` | String | Unique identifier for the Jira task |
| `Status` | String | Current state of the issue |
| `Story Points` | Number | Assigned complexity value |
| `Sprint` | String | Sprint name/ID |
| `Created Date` | DateTime | Timestamp when the issue was created |
| `Resolved Date` | DateTime | Timestamp when the issue was closed |
