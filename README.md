# Agile Delivery Risk Analytics

## Problem

Teams fail to predict sprint delivery risk early, leading to missed deadlines and inefficiencies.

## Solution

This project builds a risk analytics system using Excel, Power BI, and Jira to detect sprint failure risk mid-cycle.

## Tools

* Excel
* Power BI
* Jira

## Project Structure

| Folder | Description |
| :--- | :--- |
| `data/` | Raw and processed datasets |
| `excel-model` | Quantitative risk modeling and calculations |
| `powerbi-dashboard` | Interactive visual analytics and risk tracking |
| `jira-export` | Raw data extracts from Jira |
| `docs/` | Business and functional documentation |
| `reports/` | Executive insights and project summaries |
| `assets/` | Media, icons, and supporting visuals |
| `logs/` | Project tracking and execution logs |
